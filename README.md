# 📰 Vietnamese News Classification

## 📌 Introduction

This project focuses on building a machine learning model to automatically classify Vietnamese news articles into different categories.
It aims to demonstrate text preprocessing, feature extraction, and classification techniques in Natural Language Processing (NLP).

---

## 🚀 Technologies Used

* Python
* Scikit-learn
* Pandas, NumPy
* Matplotlib
* NLP (Text Processing)

---

## 📊 Dataset

* Vietnamese news dataset (CSV format)
* Includes multiple categories such as:

  * Politics
  * Sports
  * Technology
  * Entertainment

---

## ⚙️ Features

* Data preprocessing (cleaning, tokenization)
* Text vectorization (TF-IDF / CountVectorizer)
* Model training with machine learning algorithms:

  * Naive Bayes
  * Random Forest
  * (Optional) Neural Network
* Model evaluation:

  * Accuracy
  * Confusion Matrix
* Prediction on new unseen data

---

## 🧠 Workflow

1. Load dataset
2. Clean and preprocess text
3. Convert text to numerical features
4. Train classification model
5. Evaluate performance
6. Predict new samples

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python train.py
```

---

## 📈 Results

* Achieved high accuracy on test dataset (depending on model and preprocessing)
* Demonstrates effectiveness of ML in Vietnamese text classification
